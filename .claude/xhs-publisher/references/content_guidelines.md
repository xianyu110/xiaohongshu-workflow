# 小红书内容规范与最佳实践

## 内容要求

### 标题规范（重要）
- **最大长度：20个字**（这是硬性要求）
- 标题要吸引人，简洁明了
- 可以使用emoji增加吸引力
- 避免使用过多标点符号

### 正文规范
- 分段清晰，便于阅读
- 可以使用emoji作为分隔符或重点标记
- 长度适中，一般300-1000字为佳
- 避免过度营销化的语言

### 话题标签
- 添加相关话题标签能增加曝光
- 建议3-5个标签
- 标签要与内容相关
- 格式：#话题名称

## 媒体内容要求

### 图片
- **支持格式**：.jpg, .jpeg, .png, .gif, .webp, .bmp
- **推荐尺寸**：3:4（竖图）或 1:1（方图）
- **数量限制**：1-9张
- **建议**：
  - 图片清晰，画质高
  - 第一张图最重要，决定点击率
  - 竖图比横图更适合小红书的展示

### 视频
- **支持格式**：.mp4, .mov 等
- **推荐尺寸**：9:16（竖屏）
- **时长建议**：15秒-5分钟
- **文件大小**：建议不超过 1GB

## 发布策略

### 发布时间
- **黄金时段**：
  - 早上 7:00-9:00（通勤时间）
  - 中午 12:00-14:00（午休时间）
  - 晚上 18:00-23:00（下班后）
- 周末流量通常高于工作日

### 发布频率
- 每天最多50篇（系统限制）
- 建议每天1-3篇，保持稳定更新
- 避免短时间内大量发布

### 内容类型优先级
1. 图文内容（推荐，流量较好）
2. 视频内容
3. 纯文字内容（流量相对较低）

## 常见内容类型

### 1. 探店/美食
- 标题：地点 + 特色
- 内容：环境、味道、价格、推荐菜品
- 图片：环境照 + 菜品特写

### 2. 穿搭/时尚
- 标题：场合 + 风格
- 内容：搭配理念、单品来源、价格
- 图片：全身照 + 细节照

### 3. 护肤/美妆
- 标题：产品 + 效果
- 内容：使用感受、适合肤质、购买渠道
- 图片：产品照 + 使用前后对比

### 4. 旅行/攻略
- 标题：目的地 + 亮点
- 内容：行程安排、花费、注意事项
- 图片：风景照 + 实用信息截图

### 5. 干货/教程
- 标题：问题 + 解决方案
- 内容：步骤详解、注意事项
- 图片：步骤图 + 效果图

## 提升流量的技巧

1. **封面图**：第一张图要吸引眼球
2. **标题**：使用疑问、数字、对比等技巧
3. **标签**：选择热门且相关的标签
4. **互动**：及时回复评论，增加互动率
5. **时机**：在黄金时段发布
6. **质量**：内容原创、图片清晰、信息有价值

## 注意事项

### 避免违规
- 不要发布违法违规内容
- 不要过度营销和广告
- 不要使用他人作品未经授权
- 不要发布虚假信息

### 账号安全
- **重要**：同一账号不要在多个网页端登录
- 登录后使用移动App查看内容
- 避免频繁修改账号信息
- 遵守平台规则，避免被限流

## 内容风格建议

### 口吻
- 真诚、自然、接地气
- 避免过于官方或生硬
- 可以适当使用网络流行语
- 保持个人风格

### 结构
- 开头引起兴趣
- 中间详细说明
- 结尾总结或互动
- 善用emoji和换行
